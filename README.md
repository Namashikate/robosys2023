# robosys2023
ロボットシステム学の練習リポジトリ
# インストール方法
* ターミナルを開いた後、以下のコマンドにてクローニングを行います.
``` 
git clone https://github.com/Namashikate/robosys2023.git 
```
* クローニング後、ディレクトリ"robosys2023"に移動します.
``` 
cd robosys2023 
```
